Interactive Scenario Practice for New Scribes

Welcome

This website is designed to help new scribes practice writing complete and clinically accurate History of Present Illness (HPI) sections. Use the scenarios below to simulate patient encounters, listen to audio, and write an HPI using a provided template. After each submission, you can compare your HPI to a model version with highlights of pertinent information.

How to Use This Website

  • Select one of the six scenarios below to begin.
  • Click the Start Audio button to listen to the scenario — you will not be able to pause.
  • Use the template provided to write your HPI.
  • Use the F2 key to quickly navigate to each asterisk placeholder.
  • Submit your HPI and compare it to the original version with key details highlighted.
  • You may retry scenarios as many times as needed.

HPI Writing Guide

Please refer to the detailed HPI examples available in the ScribeNest Documentation Training on Paycom. Below is a summary of the elements that make up a strong HPI:

Essential Elements:

  • Chronology: When did symptoms start? Was the onset sudden or gradual?
  • Location: Where is the pain or complaint?
  • Quality: Describe the pain (e.g., dull, sharp, cramping).
  • Timing: Is it constant or intermittent?
  • Severity: Mild, moderate, severe? Pain scale 0–10?
  • Modifying Factors: What improves or worsens the symptoms?
  • Associated Symptoms: What other symptoms are present or denied (e.g., nausea, cough)?

Recommended HPI Structure:

  • Start with the patient’s name, age, and sex (auto-filled in the template).
  • Include relevant past medical history (PMHx) and surgical history (PSHx).
  • State the arrival mode (e.g., EMS, private vehicle) and accompanying individuals (e.g., family, EMS).
  • Provide a clear description of the chief complaint with timing, location, quality, and severity.
  • Add associated symptoms that are either endorsed or denied.
  • Include any additional relevant info shared by the patient, EMS, or others.
